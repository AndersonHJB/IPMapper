# IPMapper

你好，我是悦创。

根据 IP 地址获取地理位置信息。

这是一个基于Flask的开源IP地址地理位置查询 API，可以部署在 GitHub、Vercel 或其他服务器上。


## 部署到Vercel

你可以点击下面的按钮一键部署此项目到Vercel。
